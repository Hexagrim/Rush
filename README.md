# Rush

*A 30min 2d platformer about just platforming like celeste and ori and hollowknight.*

## About

Rush is a small platformer inspired by movement heavy games like celeste but better!  
The game is built around unitys force based controls and a not so challenging traversal rather than combat with slight hint of metroidvaniaishh.

Every area introduces new movement challenges and tests your mastery of the core abilities.

## Controls

| Key | Action |
|---|---|
| A / D | Move Left / Right |
| W | Jump |
| E | Dash |
| S | Slam |
| A-D on wall + W | Wall Slide + Wall Jump |

## Main Abilities

- Wall Slide / Wall Jump
- Dash
- Double Jump
- Slam *(optional :>)*

## Areas

- Green Grass Area
- Dusty Sand Area
- Stone Area
- Amethyst Area
- End Void Area
